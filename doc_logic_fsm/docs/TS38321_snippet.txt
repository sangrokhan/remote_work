5.1.1 Random Access procedure initialization
The RRC entity triggers the Random Access procedure by providing to the MAC entity:
1> the Random Access Config;
1> the Preamble Index and PRACH resource;
The MAC entity shall:
1> flush the Msg3 buffer;
1> set the PREAMBLE_TRANSMISSION_COUNTER to 1;
1> set the PREAMBLE_POWER_RAMPING_COUNTER to 1;
1> perform the Random Access Resource selection procedure.

5.1.4 Random Access Preamble transmission
The MAC entity shall, for each preamble:
1> instruct the physical layer to transmit a preamble using the selected PRACH;
1> set the PREAMBLE_TRANSMISSION_COUNTER to PREAMBLE_TRANSMISSION_COUNTER + 1;

5.1.5 Reception of the Random Access Response
If the Random Access Response is successfully received:
1> the MAC entity shall consider the Random Access procedure successfully completed;
1> indicate the reception of the Random Access Response to upper layers.
