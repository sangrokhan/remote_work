LangGraph Vis Week 04

백엔드 API + SSE 스트림 데모 페이지

사용 방법

  1. 아래 Backend URL에 현재 실행 중인 서버 주소를 넣습니다.
  2. runId와 threadId를 넣습니다.
  3. Load state → /api/runs/{runId}/state 조회
  4. Load events → /api/runs/{runId}/events 조회
  5. Open stream → SSE /api/runs/{runId}/events/stream 구독

주의

  • 조회 전용만 쓰던 흐름에서 Create run과 Emit event를 통해 실시간 입력 데모로 확장했습니다.
  • 입력한 runId가 존재하지 않으면 404가 나옵니다.
  • runId는 영문 소문자 시작, 소문자/숫자/_,- 조합이어야 합니다.

API 문서: /docs , 루트 접속: / → /ui/
