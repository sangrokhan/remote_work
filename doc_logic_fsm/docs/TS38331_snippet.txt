5.3.3.3 Reception of the RRCSetup by the UE
The UE shall perform the following actions upon reception of the RRCSetup:
1> stop timer T300, if running;
1> if the RRCSetup is received in response to an RRCSetupRequest:
2> discard the cellGroupConfig, if any;
2> stop timer T319, if running;
2> stop timer T302, if running;
2> stop timer T304, if running;
2> enter RRC_CONNECTED state;
2> set the content of RRCSetupComplete message as follows:
3> set the selectedPLMN-Identity to the PLMN selected by upper layers;
3> set the registeredAMF to the value received from upper layers, if available;
2> submit the RRCSetupComplete message to lower layers for transmission;
1> stop timer T320, if running;
1> stop timer T325, if running;
1> consider the current cell to be the PCell;
1> the procedure ends.
