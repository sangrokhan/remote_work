# Remote Work & Research Lab 🧪

This directory contains various independent research projects, experimental tools, and proof-of-concept implementations.

## 📂 Sub-Projects

### [A2A (Agent-to-Agent)](A2A/)
- **Description**: A framework for multi-agent communication and task delegation.
- **Key Features**: Local LLM serving (vLLM/Ollama), Summarizer and Emailer agent workflows.
- **Tech**: Python, Docker, vLLM.

### [doc_logic_fsm](doc_logic_fsm/)
- **Description**: Automated logic extraction from technical specifications (like 3GPP TS) into Finite State Machines (FSM).
- **Key Features**: NLP-based logic parsing, FSM synthesis, and visualization.
- **Tech**: Python, SpaCy, Graphviz.

### [graph_3gpp](graph_3gpp/)
- **Description**: Knowledge graph construction pipeline for 3GPP documentation.
- **Key Features**: Document chunking, LLM-based triple extraction, and Neo4j storage.
- **Tech**: Python, Neo4j, LangChain.

### [stat_graph_vis](stat_graph_vis/)
- **Description**: Statistical correlation network visualization tool.
- **Key Features**: Visualizes relationships between metrics (e.g., college major outcomes) as a weighted graph.
- **Tech**: Python, Pandas, NetworkX/Pyvis.

### [lora_examples](lora_examples/)
- **Description**: Practical examples of LoRA (Low-Rank Adaptation) training and inference.
- **Key Features**: Basic training, LoRA merge/separate inference, and custom forward hooks.
- **Tech**: Python, PyTorch.

### [DA (Data Analysis)](DA/)
- **Description**: General-purpose data analysis library and scripts.
- **Tech**: Python, Matplotlib, Seaborn.

### [transfer_learning](transfer_learning/)
- **Description**: Demo and utilities for transfer learning between models.
- **Tech**: Python, PyTorch.

### [graph_excel](graph_excel/)
- **Description**: Tools for processing Excel data into graph formats.
- **Tech**: Python, OpenPyXL.
